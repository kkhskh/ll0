What Each File Contains
Each .nc file (like 14009.nc, 15007.nc, etc.) is a data cube for one specific wildfire event that includes:

Sentinel-2 multispectral imagery (12 bands at 10-20m resolution)
Sentinel-1 SAR data (radar imagery, cloud-penetrating)
Meteorological data from NASA Power (temperature, humidity, precipitation, wind speed, soil moisture, etc.)
Burned area mask (ground truth labels from EFFIS)
Event metadata (coordinates, date, size)

File Naming System
The numbers in the filenames (14009, 15007, etc.) are unique event IDs corresponding to specific wildfire events from the EFFIS database between 2018-2022.
Why NetCDF Format
NetCDF is perfect for this type of multi-dimensional satellite data because it can store:

Multiple spectral bands
Time series data
Geospatial coordinates
Metadata
Multiple data types (imagery + weather + labels) in one file
