sed on my analysis of this research paper on machine learning for volcanic deformation detection, here are the key datasets and their sources:
Primary Datasets Used
1. Sentinel-1 SAR Data

Source: European Sentinel-1 constellation offers global coverage of radar images with a repeat cycle of 6 days at best, amounting to >13 TB per day
Access Point: Sentinel-1 data is taken directly from the Centre for Environmental Data Analysis (CEDA) Sentinel Mirror Archive (SMA)
Coverage: 592,224 wrapped interferograms of 1084 volcanoes from November 2015 to November 2020

2. COMET-LiCSAR Processed Data

Primary Portal: COMET-LICSAR archive of processed Sentinel-1 radar images (http://comet.nerc.ac.uk/COMET-LiCS-portal/)
Processing System: COMET-LICSAR automated processing system which is based on the GAMMA SAR processing software package and runs on the UK Centre for Environmental Data Analysis infrastructure JASMIN

3. Volcano Location Data

Source: Global Volcanism Programme (Global Volcanism Program 2013) generating 592,224 wrapped interferograms of 1084 volcanoes

4. Topographic Data

Source: elevation data primarily from the Shuttle Radar Topography Mission (SRTM)

5. Atmospheric Correction Data

Source: COMET-GACOS system (Yu et al. 2018) which is based on the operational HRES weather model produced by the European Centre for Medium-Range Weather Forecasts (ECMWF) data

Real-time Monitoring Portal
The machine learning algorithms described here have been adapted to run on the open-access, web-based COMET Volcano Portal (https://comet.nerc.ac.uk/comet-volcano-portal/)
Key Dataset Characteristics

Scale: ~600,000 images covering >1000 volcanoes over 5 years
Global Coverage: ~ 10% of the interferograms are located over European volcanoes (compared to 43% in Anantrasirichai et al. (2018)) and there is good coverage of Latin American (28%) and Asian (22%) volcanoes
Detection Threshold: The detection threshold for the whole dataset is 5.9 cm, equivalent to a rate of 1.2 cm/year over the 5-year study period

pythondef detect_deformation(interferogram_path, model):
    """
    Apply trained CNN to detect deformation
    """
    # Preprocess interferogram
    patches = preprocess_interferogram(interferogram_path)
    
    # Get predictions for each patch
    probabilities = []
    for patch in patches:
        prob = model(patch)
        probabilities.append(prob)
    
    # Merge probabilities with Gaussian weights
    merged_prob = merge_probabilities(probabilities)
    
    # Apply threshold (P > 0.5)
    detection_flag = merged_prob > 0.5
    
    return merged_prob, detection_flag
